Calculadora Simples em Python
Este projeto consiste em uma calculadora simples desenvolvida em Python, que permite realizar operações básicas de adição, subtração, multiplicação e divisão entre dois números fornecidos pelo usuário.

Descrição Detalhada
A Calculadora Simples em Python é uma aplicação de linha de comando que oferece uma interface intuitiva para realizar operações matemáticas básicas. Ela foi desenvolvida utilizando a linguagem de programação Python, oferecendo funcionalidades essenciais para cálculos simples.

Funcionalidades
A calculadora oferece as seguintes funcionalidades:

Adição: Soma dois números.
Subtração: Subtrai um número de outro.
Multiplicação: Multiplica dois números.
Divisão: Divide um número pelo outro. Evita divisão por zero.
Instruções de Instalação e Uso

Python instalado na sua máquina. Você pode baixar e instalar a partir do site oficial.

Faça o download do código-fonte da calculadora ou copie o código para um arquivo .py.
Salve o arquivo em um diretório de sua escolha.
Uso
Abra um terminal ou prompt de comando.
Navegue até o diretório onde o arquivo da calculadora está localizado.
Execute o arquivo .py usando o Python:

python calculadora.py
Siga as instruções exibidas no terminal para realizar as operações desejadas.
Exemplo de Uso


Selecione a operação:
1. Adição
2. Subtração
3. Multiplicação
4. Divisão
Digite a sua opção (1/2/3/4): 1
Digite o primeiro número: 5
Digite o segundo número: 3
Resultado: 8.0
